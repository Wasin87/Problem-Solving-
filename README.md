# JavaScript Problem Solving Task

This repository contains solutions to 10 basic JavaScript problems using plain JavaScript.

## Rules Followed
- No frameworks or libraries used
- Only basic JavaScript syntax
- Own logic applied
- Each problem solved in a separate file

## Problems Covered
1. Reverse a String
2. Count Vowels in a String
3. Palindrome Check
4. Find Maximum Number
5. Remove Duplicates from Array
6. Sum of Array Numbers
7. Find Even Numbers
8. Capitalize First Letter
9. Factorial of a Number
10. PingPong Challenge
